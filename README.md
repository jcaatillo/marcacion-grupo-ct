# Marcación Grupo CT

Sistema de control de asistencia y gestión de personal para Grupo Castillo Torres S.A.

## Estructura del repositorio

Este repositorio está organizado de la siguiente manera:

- **[`/web`](./web)**: Directorio principal que contiene la aplicación web (Next.js 16 + Supabase).
- **[`/web/docs`](./web/docs)**: Documentación técnica y archivos históricos del proyecto.
- **[`/web/scripts`](./web/scripts)**: Herramientas de mantenimiento y utilidades.

Para comenzar con el desarrollo, consulta el [README de la aplicación web](./web/README.md).
