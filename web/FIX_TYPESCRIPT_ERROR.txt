# FIX: TypeScript Type Incompatibility in TabsInternal

## Error Encontrado
Error de TypeScript en Vercel build:
- Archivo: app/(admin)/employees/[id]/edit/employee-edit-form.tsx:96
- Problema: Type 'Dispatch<SetStateAction<TabId>>' is not assignable to type '(tabId: string) => void'

## Solución Aplicada
Cambiar de:
```tsx
onTabChange={setActiveTab}
```

A:
```tsx
onTabChange={(tabId) => setActiveTab(tabId as TabId)}
```

## Explicación
- TabsInternal espera una función con tipo: (tabId: string) => void
- setActiveTab tiene tipo: Dispatch<SetStateAction<TabId>>
- La solución es envolver setActiveTab en una función callback que haga el type casting
